python：windows系统下python32位与64位共存记录

发表 2022-05-16 分类 python 评论： 阅读：

在windows系统下如何实现python32位与64位共存？不同版本pip如何共存？

因为毕设项目需要安装python一个库，结果网上一看，这个库只支持64位python版本。再瞅瞅我的，32位，懵……

上网搜了一下，都说是先卸载32位（包括已安装的库），再安装64位。

但我32位python也用了好几年时间了，陆陆续续不知道安装了多少库，直接卸载实在可惜！并且有些库只支持32位不支持64位，若以后遇到只支持32位的库，我再把64位版本卸了安装32位的？

于是便想到同时安装32位和64位版本，经过探索具体如下。

这里假设已经实现安装好了一个python和pip版本，就假设32位的吧。

1.首先进入安装文件，修改python位python32。

2.进入cmd，输入python32 -m pip install --upgrade pip --force-reinstall重新更新pip文件。

3.修改python32位安装目录下Scripts/pip与Scripts/pip3等等等等文件名为Scripts/pip_32与Scripts/pip3_32。

4.在cmd中输入python32，发现运行的为32位版本python，输入pip_32 -v，发现运行的为32位版本python对应下的pip。

5.正常安装python64位版本，注意安装位置手动修改为自己选的位置，同时勾选安装pip、添加环境变量。

6.修改python64位安装目录下Scripts/pip与Scripts/pip3等等等等文件名为Scripts/pip_64与Scripts/pip3_64。

7.在cmd中输入python64，发现运行的为64位版本python，输入pip_64 -v，发现运行的为64位版本python对应下的pip。

以后就能通过pip_32 install与pip_64 install自由选择pip版本啦~

python32 python64 pip
